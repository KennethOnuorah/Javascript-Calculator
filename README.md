# ➕➖ JS Calculator ✖️➗
A basic calculator for performing simple mathematical operations.

## Languages used
* HTML
* CSS
* Javascript

## Features
* Four basic operators
* Operation chaining
* Decimals
* Negative sign toggling
* Square rooting
* Custom exponential expressions
* Automatic comma formatting
* Error handling
* Sound effects
